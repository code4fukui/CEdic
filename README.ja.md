# CE典 - 中英単語辞書

[CE典](https://code4fukui.github.io/CEdic/)は、中英辞典Webアプリケーションです。

## 機能
- 中国語-英語の単語定義の検索と表示
- ピンイン読みの表示
- オープンソースの「CC-CEDICT」辞書データを利用

## データ / API
辞書データは、オープンソースプロジェクトの「[CC-CEDICT](https://cc-cedict.org/editor/editor.php?handler=Main)」から取得しています。CC BY-SAライセンスで提供されています。
