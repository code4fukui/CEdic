# CEdic - 中国語-英語辞書

[CE典](https://code4fukui.github.io/CEdic/) は、中国語-英語辞書のWebアプリケーションです。

## 機能
- 中国語-英語の単語の定義の検索と表示
- ピンインでの発音表示をサポート
- オープンソース辞書「CC-CEDICT」のデータを収録

## データ / API
辞書データはオープンソースプロジェクトである[CC-CEDICT](https://cc-cedict.org/editor/editor.php?handler=Main)のものを利用しており、CC BY-SAライセンスの下で提供されています。
